About Me

Hello! I'm Mickey, a passionate web developer with a love for creating beautiful, functional digital experiences. My journey into tech began when I first discovered how to customize my MySpace page (yes, I'm dating myself here!) and I've been hooked on coding ever since.

Currently specializing in front-end development, I enjoy bringing designs to life with clean, semantic HTML and CSS. When I'm not battling with CSS specificity or JavaScript quirks, you can find me exploring new frameworks, contributing to open-source projects, or mentoring beginners in online coding communities.

What excites me most about web development is how it combines creativity with problem-solving. Every project is a new puzzle to solve, and I love that moment when everything finally clicks into place. My approach focuses on building accessible, performant websites that deliver real value to users.

Outside of coding, I'm an avid photography enthusiast and board game collector. I believe these hobbies help me think differently about user experiences and interface design. I'm always looking to connect with fellow developers and creative professionals, so don't hesitate to reach out!
